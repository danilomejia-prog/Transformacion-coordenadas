Transformador UTM WGS84 → PSAD56

Versión para GitHub Pages con importación y exportación CSV. Incluye zonas continentales de Ecuador y las zonas UTM de Galápagos.

Continente: 17S, 18S Galápagos: 15S, 16S Cerca del ecuador: N o S según el dato

1. Cargar CSV

2. Configurar columnas

El archivo de salida agregará las columnas: Este_PSAD56, Norte_PSAD56, EPSG_Origen, EPSG_Destino y Estado_Transformacion.

3. Procesar y descargar

Nota técnica importante: Esta versión incluye Galápagos con zonas UTM 15S y 16S, porque el EPSG registra que WGS 84 / UTM 15S y 16S cubren Ecuador - Galápagos. Para Ecuador continental, WGS 84 / UTM 17S y 18S están registradas para Ecuador, y PSAD56 / UTM 17S y 18S se usan en Ecuador continental. Para Galápagos, aquí se usa una definición UTM/PSAD56 personalizada por zona para mantener la transformación en la app; conviene validar los resultados con puntos de control antes de usarla en cartografía oficial.
